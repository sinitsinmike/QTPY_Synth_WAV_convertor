Boot-режим для загрузки WAV (QTPY_Synth_V3)

В этой версии (QTPY_Synth_V3) добавлен простой “режим загрузки файлов”: зажми единственную
кнопку на девайсе и включи питание — в этом режиме CIRCUITPY-диск будет виден на 
компьютере, и ты сможешь спокойно закинуть свои wavetable WAV в папку /wav (или обновить 
любые файлы проекта). После копирования файлов просто сделай power cycle (выключил/включил 
уже без зажатой кнопки) — устройство вернётся в обычный “performance mode”, где диск 
спрятан и работает сохранение патчей.

Обновление прошивки проекта из ZIP (Git / QTPY_Synth_V3)

На Git лежит архив QTPY_Synth_V3.zip (одноимённый релиз/папка).
Скачай ZIP, распакуй, затем скопируй содержимое на диск CIRCUITPY с заменой файлов.
Папку /wav можно не трогать, если хочешь сохранить свои собственные wavetable-файлы: 
просто не перезаписывай её (или заранее сделай копию, если не уверен).
